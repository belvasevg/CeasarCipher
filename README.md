# CeasarCipher
Пример программы по шифровке/дешифровке сообщений на основе шифра Цезаря.
В main.py производится основной запуск программы.
auxiliary_functions.py содержит в себе вспомогательные функции, которые возвращают значения в функции шифровки и дешифровки.
decrypt.py используется для дешифровки сообщения с заданным значением сообщения и шага (сдвига) шифра.
encrypt.py используется для дешифровки сообщения с заданным значением сообщения и шага (сдвига) шифра.
